# Fire Detection and Emergency Response System

This Arduino-based system uses the following components:
- Smoke Sensor (MQ2)
- TMP36 Temperature Sensor
- Photoresistor (LDR)
- Red & Green LEDs
- Buzzer
- LCD 16x2 Display
